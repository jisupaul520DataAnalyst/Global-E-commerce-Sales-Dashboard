# Global E-commerce Sales Dashboard

## Project Overview
This Power BI dashboard analyzes global e-commerce sales performance using interactive visualizations and KPI metrics.

The dashboard provides insights into:
- Total Sales
- Total Profit
- Order Quantity
- Discounts
- Monthly Sales Trends
- Regional Sales Performance
- Product-wise Sales Analysis
- Category Distribution
---
## Dashboard Features

### KPI Cards
- Total Sales: 486K
- Total Profit: 160K
- Total Orders: 2000
- Total Discounts: 17K

### Visualizations
- Month-wise Sales Trend
- Top 10 Product-wise Sales
- Sales by Region
- Category Distribution

### Filters
- Region Filter
- Payment Method Filter
